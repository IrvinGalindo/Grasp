### GRASP (Greedy Randomized Adaptative Search Procedure)
----
Es un heurística basada en un procedimiento de búsqueda iterative y aleatorio el cual garantiza una Buena solución, aunque a veces esta no es óptima. 

Fases:

1. Construcción: 

     * Se forma una LC con los elementos.
     * Se crea un RLC  con los mejores elementos. 
     * Se va eligiendo un elemento en cada iteración para crear solución parcial.

2. Mejora:  Se hace una búsqueda local a partir de la solución construida hasta que no se pueda mejorar más.
